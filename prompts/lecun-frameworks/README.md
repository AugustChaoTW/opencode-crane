# Yann LeCun-Inspired AI Reasoning Prompt Frameworks

This collection provides eight specialized reasoning frameworks inspired by the core principles of Yann LeCun's vision for autonomous AI agents. These frameworks prioritize causal modeling, world representation, hierarchical planning, and energy-based optimization to guide LLMs toward more grounded and systematic decision-making.

## Selection Matrix

| Framework Name | Best For | Complexity | Output Type | When to Use |
| :--- | :--- | :--- | :--- | :--- |
| **World Model Reasoning Engine** | Causal simulation | High | Scenario Map | Predicting consequences and handling uncertainty. |
| **Multi-Step Planning Framework** | Execution strategy | High | Action Plan | Breaking down complex goals into sequential tasks. |
| **Energy-Based Decision Analyzer** | Global optimization | Medium | Trade-off Analysis | Evaluating options against multiple constraints. |
| **Self-Supervised Learning Knowledge Builder** | Deep understanding | Medium | Knowledge Synthesis | Validating comprehension and identifying knowledge gaps. |
| **Hallucination Destroyer** | Fact verification | High | Evidence Report | Separating claims from ground truth with confidence flags. |
| **Hierarchical Reasoning Engine** | Abstract strategy | High | Strategic Road Map | Moving between high-level vision and granular operations. |
| **Adversarial Thinking Simulator** | Risk mitigation | Medium | Stress Test | Testing strategies against failures and black swan events. |
| **First Principles Deconstructer** | Innovation | Medium | Foundation Map | Rebuilding solutions from ground-truth assumptions. |

## How to Use This Collection

These frameworks aren't just templates. They're cognitive architectures. To get the most out of them, follow these steps:

1.  **Identify your primary bottleneck.** Is your problem a lack of planning (#2), a need for deeper understanding (#4), or a risk assessment issue (#7)?
2.  **Select the corresponding framework.** Navigate to the individual framework directory to access the prompt text and implementation guide.
3.  **Provide rich context.** These frameworks work best when you feed them specific data points, constraints, and goals.
4.  **Iterate on the output.** Use the structured feedback from one framework to refine your input for another.

## Framework Combinations

Combine these frameworks to handle multi-dimensional challenges.

*   **The Strategic Launch Pad (#6 + #2):** Use the Hierarchical Reasoning Engine to set the vision, then pass the operational goals to the Multi-Step Planning Framework for execution.
*   **The Bulletproof Decision (#3 + #7):** Use the Energy-Based Decision Analyzer to find the optimal path, then run it through the Adversarial Thinking Simulator to find hidden failure points.
*   **The Ground-Truth Innovation (#8 + #1):** Deconstruct a problem with First Principles, then use the World Model Reasoning Engine to simulate how your new solution survives in a complex environment.
*   **The Research Validator (#4 + #5):** Synthesize new knowledge with the SSL Knowledge Builder and immediately scrub it for inaccuracies using the Hallucination Destroyer.
