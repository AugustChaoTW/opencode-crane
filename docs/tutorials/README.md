# Tutorials Index

Start here if you want the fastest path to CRANE.

## Quickstarts

- [01. Literature Review in 10 Minutes](./01-literature-review.md)
- [02. Pre-Submission Check in 5 Minutes](./02-submission-check.md)

## What each tutorial covers

- **Literature review**: search, add, download, read, and annotate papers.
- **Submission check**: run the integrated pre-publication verification flow.
- **Team research**: coordinate work with GitHub Issues and progress comments.

## Shared troubleshooting

- If you see `module not found`, run: `uv sync`.
- If `crane` commands are unavailable, make sure the repo dependencies are installed.
- If GitHub commands fail, run `gh auth status`.

## Suggested path

1. Run the literature review quickstart.
2. Add a few papers and annotations.
3. Run the submission check.
4. Coordinate remaining work with issues.
