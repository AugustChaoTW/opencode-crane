# v0.10.1: Journal-Aware Writing Style Toolkit

## Overview

Intelligent writing style optimisation that learns from journal exemplars and provides section-by-section diagnosis. The toolkit analyses a paper's writing style against heuristic targets derived from publisher conventions (IEEE, ACM, Springer/Nature, Elsevier) and the 55-journal Q1 database.

## Architecture

### Phase A: Data Models & Section Parsing

- **`StyleMetrics`** — 7 quantitative measurements: sentence length, vocabulary diversity, passive voice ratio, hedging frequency, citation density, formality, technicality.
- **`StyleGuide`** — Per-journal target profile with section-level metric targets and exemplar patterns.
- **`SectionDiagnosis`** — Comparison result: current vs target metrics, deviation score, issues, suggestions.
- **`RewriteSuggestion`** — Concrete rewrite with original text, suggested text, rationale, and exemplar source.
- **Section normalisation** — Maps free-form headings ("Methodology", "Experiments") to 7 canonical types.
- **`compute_metrics()`** — Strips LaTeX, splits sentences, computes all 7 metrics via regex-based NLP.

### Phase B: Service Layer

- **`WritingStyleService`** — Core diagnostic engine initialised with a journal name.
  - Loads journal profile from `data/journals/q1_journal_profiles.yaml` (55 journals).
  - Resolves publisher → style class (4 classes: `ieee_transactions`, `acm`, `springer_nature`, `elsevier`).
  - Builds `StyleGuide` with section-level target metrics.
  - Detects domain via `DomainPackLoader` (AI/ML, Cybersecurity, IoT, MIS).
  - Provides: `diagnose_section()`, `diagnose_full_paper()`, `get_exemplars()`, `suggest_rewrites()`, `compare_with()`, `generate_report()`.

### Phase C: MCP Tools (7 tools)

| Tool | Purpose |
|------|---------|
| `crane_extract_journal_style_guide` | Build/retrieve style guide for any of 55 journals |
| `crane_diagnose_section` | Analyse one section against journal targets |
| `crane_diagnose_paper` | Full paper diagnosis across all sections |
| `crane_get_style_exemplars` | Retrieve exemplar writing patterns for a section |
| `crane_suggest_rewrites` | Generate rewrite suggestions based on diagnosis |
| `crane_compare_sections` | Compare style expectations across two journals |
| `crane_export_style_report` | Generate comprehensive Markdown report |

### Data Flow

```
Paper PDF/LaTeX
    ↓
parse_latex_sections() → SectionLocation[]
    ↓
compute_metrics() → StyleMetrics (current)
    ↓
WritingStyleService._build_style_guide() → StyleMetrics (target)
    ↓
_compute_deviation() + _identify_issues() + _generate_suggestions()
    ↓
SectionDiagnosis → MCP Tool response (JSON dict)
    ↓
User: Section diagnosis + exemplars + suggestions
```

## Integration with 55-Journal System

- Uses `data/journals/q1_journal_profiles.yaml` to load journal metadata (publisher, scope, IF).
- Maps publisher to style class for target metric selection.
- Applies domain packs from `src/crane/config/domain_packs/` for domain-specific evaluation.
- Supports all 55 Q1 journals across 4 domains (CS, Cybersecurity, IoT, MIS).
- Journal lookup supports both full name and abbreviation (e.g. "IEEE TPAMI").

## Style Classes

| Class | Publishers | Characteristics |
|-------|-----------|-----------------|
| `ieee_transactions` | IEEE | Formal, technical, structured contributions list |
| `acm` | ACM, JMLR | Empirical rigour, RQ-driven, statistical significance |
| `springer_nature` | Springer, Nature, MDPI | Narrative flow, comprehensive methodology |
| `elsevier` | Elsevier, Wiley, Taylor & Francis | Module-based architecture, comparative evaluation |

## Example Usage

```python
from crane.services.writing_style_service import WritingStyleService

service = WritingStyleService("IEEE TPAMI")
diagnosis = service.diagnose_section("main.tex", "Introduction")
suggestions = service.suggest_rewrites(diagnosis)

service2 = WritingStyleService("ACM TOSEM")
comparison = service.compare_with(service2, "Methods")
```

Via MCP tools:

```
crane_diagnose_paper(paper_path="main.tex", journal_name="IEEE TPAMI")
crane_get_style_exemplars(journal_name="IEEE TPAMI", section_name="Introduction")
crane_suggest_rewrites(paper_path="main.tex", section_name="Methods", journal_name="IEEE TPAMI")
crane_compare_sections(paper_path="main.tex", section_name="Introduction", journal1="IEEE TPAMI", journal2="ACM TOSEM")
```
